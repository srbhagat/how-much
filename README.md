# How Much Possible Which Side

Calculates:
- LONG / SHORT / NO TRADE
- Possible TP points
- Suggested SL points
- Entry area
- TP price
- SL price

Uses CoinDCX candles, live price, adaptive matching, ATR, support/resistance, VWAP, MTF bias, volume zone, liquidity sweep, BOS/CHOCH.

Deploy: upload app.py, requirements.txt, README.md and reboot Streamlit.
